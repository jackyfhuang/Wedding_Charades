# Wedding Charades Game

A simple browser-based charades game with wedding-related words.

## How to Play

1. Click "Get Word" to see a random wedding-related word
2. Act it out in charades!
3. Click "✓ Correct" or "✗ Incorrect" to track your score
4. The next word appears automatically
5. Click "Reset Game" to start over with all words

## Files

- `index.html` - Main HTML file
- `style.css` - Styling
- `script.js` - Game logic and word list
